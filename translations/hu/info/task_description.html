Adott egy pozitív egész szám. Önnek meg kell határozni az adott szám számjegyeinek a szorzatát, a nullákat kivéve.

Például: adott az 123405 szám. Az eredmény a következő lesz: 1*2*3*4*5=120 (ne feledje kizárni a nullákat!).

Bemeneti adatok: egy pozitív egész szám.

Kimeneti adatok: a számjegyek szorzata egy egész számként (int).

Példák: 

{% if interpreter.slug == "js-node" %}
digitsMultip(123405) == 120
digitsMultip(999) == 729
digitsMultip(1000) == 1
digitsMultip(1111) == 1
{% else %}
checkio(123405) == 120
checkio(999) == 729
checkio(1000) == 1
checkio(1111) == 1
{% endif %}

Alkalmazása: Ennek a feladatnak a segítségével meg lehet tanulni az egyszerű adattípusok közötti átalakításokat.

Előfeltételek:
0 < number < 106
